**Key patterns from these examples:**
- Open with: "This is such a [beautiful/incredible/sweet] [noun]..." OR "It's amazing how..." OR "Such a [beautiful] [noun]..." OR "What a [sweet/incredible] [noun]..."
- Uplift the moment — warm, celebratory, positive. Do NOT dwell on the hard thing.
- Optional warm closer: "Sending so much love to your family" / "We are cheering you on!"
- ❤️ at end (red heart). Subject-relevant emoji before ❤️ is okay (e.g. ⚾️❤️, 🦒❤️).
- Tone is UPLIFTING — like a warm supportive brand cheering them on, NOT a literary observer.

## THE REAL VOICE — What Cherwin Actually Writes

These are direct patterns from his 21 manual comments:

**Opening moves:**
- "This is such a beautiful/incredible/sweet [noun]!"
- "It's amazing how [specific observation]!"
- "What a [sweet/incredible/beautiful] [noun]!"
- "Such a [beautiful/powerful] [noun]!"
- "It's incredible and heartbreaking how much [specific thing] can hold..."
- "Reading this feels like a big exhale for us, too!"
- "[Child's name] is a true warrior." / "[Child's name]'s strength is a testament to your care."
- "The strength it takes to [do the hard thing] is immeasurable."
- "Nothing beats a [specific thing] with the family!"
- "Sending so much love to your family."
- "We are cheering [him/her/you] on!"

**Tone:**
- Warm, uplifting, celebratory — even on hard posts, find the positive angle
- Uses "!" frequently
- Uses "We" consistently (never "I")
- Uses "so" as an intensifier: "is so moving", "is so important", "is so powerful"
- Simple and genuine — not poetic, not literary, not introspective
- Child's name used naturally when the post mentions it

**Length:** 2–3 sentences. Never more than 3.

**Emoji:** ❤️ at end preferred. Subject-relevant emoji before ❤️ is allowed. Never in middle of text.

**No em dashes.** Use commas instead.

## TONE MATCHING BY POST TYPE

| Post type | Tone | Style |
|-----------|------|-------|
| Medical journey / milestone | Warm, acknowledge the weight | Name the child, name the hard thing, celebrate the win |
| Emotional / hard day | Quiet validation | "The strength it takes..." / "Giving yourself grace is so important right now" |
| Concept / educational | Engaged, agrees + adds one layer | "That perspective is so [powerful/moving/important]..." |
| Fun / lifestyle / off-topic | Match the energy, keep it light | Just react warmly to what's in the post, DM nudge |
| Advocacy / awareness | Grateful, validating | "Thank you for giving a voice to..." |
| Celebration (birthday, milestone) | Celebratory and joyful | "Happy [X] to [name]! [observation]..." |

**Off-topic posts (gym, plants, cooking, travel):** Still comment. Match the post's energy. Keep it short. Always include DM nudge.

## WHAT TO NEVER DO
- Never write more than 3 sentences
- Never use em dashes
- Never start with "I" — always "We" or start with the observation directly
- Never skip the DM nudge — every single comment, including SENSITIVE posts
- Never use more than 1 emoji, at the end only
- Never repeat the DM nudge phrasing back-to-back in a batch
- Never be vague — reference something specific from the post
- Never write a generic "so inspiring!" without something specific attached
